# Phase 2 - System Development & Integration

Frontend Development (Electron + React)
	•	Built a cross-platform desktop application using Electron, supporting Windows, macOS, and Linux.
	•	Developed a React-based UI for uploading unstructured requirement documents (PDF, TXT, DOCX).
	•	Implemented interactive dashboards to display extracted requirements, classifications, clusters, and priority levels.
	•	Created integration dashboards to securely connect with external platforms.
	•	Designed a report generation interface to export structured Software Requirement Specification (SRS) summaries.

Backend Orchestration & Data Integration
	•	Developed a Spring Boot RESTful API to manage communication between frontend and backend services.
	•	Implemented secure authentication and authorization using the Atlassian OAuth2 login flow.
	•	Integrated the Jira API to fetch accessible resources and project-related information.
	•	Configured a secure RestClient with automatic Bearer Token injection for Atlassian API requests.
	•	Built a Unified Text Formatting module to clean, normalize, and process raw communication data from external sources.
